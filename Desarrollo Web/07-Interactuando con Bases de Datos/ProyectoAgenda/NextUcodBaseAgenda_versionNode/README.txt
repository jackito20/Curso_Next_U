INSTALACION:
Verison de Node Js usada: 8.9.4

1.- Desde la consola del sistema ubicarse en la carpeta raíz del proyecto: codBase
2.- ejecutar npm install
3.- En el archivo codBase/Server/new_user.js, configurar el usuario que ingresara al sistema
4.- desde la consola del sistema ubicarse en la carpeta Server
5.- Ejecutar node new_user.js (creara el usuario y base de datos mongo)
7.- Ejecutar el servicio mongod
6.- Desde la carpeta codBase iniciar la ejecucion del servidor con el comando: node server/index.js
7.- Ingregar al sistema con la url: http://localhost:3000
8.- Ingresar con el usuario y contraseña configurados en el paso 3
